4\. Data Transformation (Make Data Learnable)
---------------------------------------------

**Goal:** Convert data into a form suitable for ML algorithms.

### Sub-steps

#### a. Feature Scaling

*   Normalization (0–1 range)
    
*   Standardization (mean = 0, std = 1)
    
*   Robust scaling (for outliers)
    

#### b. Encoding Categorical Data

*   Label encoding
    
*   One-hot encoding
    
*   Target encoding
    

#### c. Data Type Conversion

*   Text → numerical
    
*   Date → components (day, month, year, weekday)
    

#### d. Mathematical Transformations

*   Log transform
    
*   Box-Cox / Power transform
    

**Outcome:** Numerical, well-scaled inputs.
