2\. Data Cleaning (Remove Noise & Errors)
-----------------------------------------

**Goal:** Make data accurate and consistent.

### Sub-steps

*   Handle missing values
    
    *   Remove rows/columns
        
    *   Impute (mean, median, mode, model-based)
        
*   Remove duplicate records
    
*   Correct inconsistent labels (e.g., `M`, `Male`, `male`)
    
*   Fix invalid values (negative age, impossible dates)
    
*   Handle outliers
    
    *   Capping
        
    *   Removal
        
    *   Transformation
        

**Outcome:** Trustworthy data.
